# Sentiment Analysis of X (Twitter) Data
## Project Overview

This project focuses on performing sentiment analysis on tweets collected from X (formerly Twitter). The objective of the project is to analyze user opinions and classify them into positive, neutral, and negative sentiments. Sentiment analysis helps in understanding public opinion, user behavior, and overall sentiment trends on social media platforms.

Dataset Information

The dataset contains tweets along with sentiment labels. The sentiment categories used in this project are:

* Positive (1)
* Neutral (0)
* Negative (-1)

The dataset was analyzed using Python and various data analysis libraries.
Tools and Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

Project Workflow

1. Data Loading
2. Data Exploration
3. Sentiment Distribution Analysis
4. Data Visualization
5. Findings and Interpretation
6. Conclusion

Visualizations Created

* Sentiment Distribution Count Plot
* Positive, Neutral, and Negative Tweets Analysis

Key Findings

* Positive tweets were the most common sentiment in the dataset.
* Neutral tweets represented a significant portion of the data.
* Negative tweets were comparatively fewer.
* Sentiment analysis provides valuable insights into public opinion and social media trends.

Files Included

* Sentiment_Analysis_X_Data.ipynb
* Sentiment_Analysis_X_Data_Report.pdf

Conclusion

This project demonstrates how sentiment analysis can be used to understand public opinion through social media data. By utilizing Python libraries such as Pandas, Matplotlib, and Seaborn, meaningful insights were extracted and visualized effectively.
